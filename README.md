# Bayesian Spatio-temporal Modelling Demo App
This R-shiny application allows user to apply to Bayesian Spatio-temporal Models to ESSENCE data through the [epistemic package](https://gitlab.jhuapl.edu/nationalhealth/essence/essence-ai-toolkit/epistemic).  It includes tools to generate basic queries, define custom models, fit models and save models, and visualize model outputs through interactive tables, timeseries plots and maps.
